# Oil Risk Analysis

An interactive data analysis project focused on identifying, understanding, and visualizing risks related to oil markets, routes, and supply disruptions.  
This project combines data analysis, visualization, and risk-focused insights to help explore how geopolitical, logistical, and market factors can impact oil transportation and pricing.

## Project Overview

The Oil Risk Analysis project is designed to study oil-related risk factors such as:

- Supply chain disruptions
- Route-based transportation risks
- Geopolitical tensions
- Price volatility
- Region-wise oil movement patterns
- Risk scoring and trend analysis

The goal is to transform raw data into meaningful insights using Python-based analysis and visualizations.

## Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Risk factor identification
- Visual analysis of oil routes and regions
- Price/risk trend visualization
- Insights for decision-making
- Notebook-based workflow for easy understanding
